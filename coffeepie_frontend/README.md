# Coffee Pie Frontend ☕

Current Version: v0.1.0<br>

Graphical User Interface: Coffee Pie Qt<br>
"Stupid simple" GUI for everyone, even without technical knowdledge, just basic alphabetization required to input credentials in the login screen and a couple clicks to start using the service.<br>

Program: Coffee Pie CLI<br>
Basic terminal interface for experienced IT Administrators usually for high scale deployments like Web Servers, Databases, Applications, Content Delivery Networks, Docker, Kubevirt, etc.<br>
Shortcut (Ctrl + Alt + T)
